========================================================================
    STATIC LIBRARY : RandomWalksLib Project Overview
========================================================================

This library implements the Random Walks for Image Segmentation algorithm
 Implementation based on the work:
 Grady, L.: Random walks for image segmentation, IEEE Transactions on Pattern Analysis and
 Machine Intelligence,28,11,1768-1783,2006

 For 2D and 3D matrices and provides various solvers for the resulting sparse, symmetric, and positive definite
 system of linear equations.

 Furthermore, the library implements the confidence estimation approach for ultrasound
 Implementaiton based on the work:
 A. Karamalis, A. Katouzian, S. Carlier, N. Navab 
 Confidence Estimation in IVUS Radio-Frequency Data with Random Walks 
 IEEE International Symposium on Biomedical Imaging, Barcelona, Spain, 2012 (bib)
